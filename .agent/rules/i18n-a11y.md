# i18n & Accessibility (a11y)

## Multilingual Support (i18n)

- The app supports English (en), Finnish (fi), and Swedish (sv).
- **All user-facing text MUST be internationalized** using `next-intl`.
- Translation files are located in `/messages/{locale}.json`.
- Add new keys to all locale files simultaneously.

## Accessibility (a11y)

- **Compliance**: Aim for WCAG 2.1 Level AA.
- **Semantic HTML**: Use proper tags (`<header>`, `<main>`, `<nav>`, `<h1>`â€“`<h6>`).
- **Interactive Elements**: Must be keyboard accessible (Tab, Enter, Space).
- **Aria Labels**: Use for icon-only buttons and to provide context for screen readers.
- **Focus Indicators**: Ensure they are visible (`focus-visible`).
- **Color Contrast**: Maintain at least 4.5:1 for text.

## Animations & UX

- Use `framer-motion` for smooth transitions.
- Keep animations under 300ms.
- Respect `prefers-reduced-motion`.
- Provide loading skeletons or shimmer effects for data-heavy components.
